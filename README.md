# scala-cli-maven-surefire-pmd-junit-test-car-data

## Description
Analyze source code for bugs.
A POC for sprimg app using JUnit
framework with surefire plugin.

## Tech stack
- scala
- maven
	- pmd
  - spring
  - junit
  - surefire

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
